# Computational Motor Control (CMC) 2026

Student Exercise Repository for the 2026 edition of the Computational Motor Control course.

## Getting Started

Before starting **Lab 0**, please follow the instructions at the following link:
[https://farmsim.gitlab.io/courses/cmc-2026-students/](https://farmsim.gitlab.io/courses/cmc-2026-students/)

## Repository Structure

Each lab session has a dedicated folder containing two subfolders:
- **`Report`**: This folder contains the assignment PDF for the lab.
- **`Python`**: This folder contains the implementation code you will need to modify to answer the assignment questions.
